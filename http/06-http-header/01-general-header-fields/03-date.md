# Date

> 2018-11-28 @wsl

表明创建HTTP报文的日期和时间。

第一种形式较为常用。

- RFC1123 格式

  HTTP/1.1 版本使用在RFC1123中规范的日期时间的格式。

  ```
  Date: Tue, 03 Jul 2012 04:40:59 GMT
  ```

-  RFC850格式

  之前的HTTP协议版本中使用在RFC850中定义的格式。

  ```
  Date: Tue, 03-Jul-12 04:40:59 GMT
  ```

- 与C标准库相似 格式

  除此之外，还有一种格式。它与C标准库内的asctime()函数的输出格式一致。

  ```
  Date: Tue Jul 03 04:40:59 2012
  ```
