### 1. 前端性能优化方案？

- 代码压缩，静态资源压缩
- 图片压缩
- CDN
- 优化代码质量减少过多的DOM操作
- 合并请求
- 提升合成层
- raf(渲染间隔16.7ms，减少回流与重绘)
- 懒加载
